# Simulador de Epidemias — Probabilidade de Infecção

![Python](https://img.shields.io/badge/Python-3.x-blue?logo=python&logoColor=white) ![Topic](https://img.shields.io/badge/Topic-simulation-green) ![License](https://img.shields.io/badge/License-MIT-yellow)

## Descrição

Simulação estocástica de disseminação de epidemias com 1000 pessoas, 30 dias e 1000 simulações independentes para calcular a probabilidade de infecção de >50% da população.

## Funcionalidades

- 1000 simulações Monte Carlo independentes
- Modelagem de 1000 indivíduos ao longo de 30 dias
- Cálculo de probabilidade de infecção em massa (>50%)
- Visualização da distribuição de casos finais
- Parâmetros configuráveis de infecção e recuperação

## Stack Tecnológico

| Tecnologia | Descrição |
|---|---|
| Python | Linguagem principal |
| NumPy | Simulações numéricas eficientes |
| Matplotlib | Visualização de resultados |

## Como Usar

1. Instale as dependências: `pip install numpy matplotlib`
2. Execute: `python simulador_epidemia.py`
3. Os resultados e gráficos serão exibidos automaticamente
